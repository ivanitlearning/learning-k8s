# Command shortcuts

| Short form     | Long form                 |
| -------------- | ------------------------- |
| -n             | --namespace               |
| -A             | --all-namespaces          |
| kubectl get cm | kubectl get configmap     |
| csr            | certificatesigningrequest |

Can view the short forms with `kubectl api-resoures --sort-by=name`
